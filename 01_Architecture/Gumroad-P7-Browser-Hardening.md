## DESCRIPTION

The browser is the largest attack surface in any system. Default settings — even on privacy-focused browsers — leak more than most people realize: IP via WebRTC, fingerprint via canvas, tracking via persistent cookies.

This config closes those gaps with exportable settings you apply once and keep.

No manual tweaking of 200 about:config entries. No half-measures. A complete, tested configuration built as Phase 3 of a hardened AI operating system — used daily under Mullvad VPN with strict DNS filtering.
